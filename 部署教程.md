# 德州扑克联机版 —— 部署上线教程

这份教程教你把联机版发布成一个网址，发给朋友点链接就能一起玩。

面向完全没部署过的人，一步步来。整个过程约 10 分钟，**全部免费**。

---

## 你拿到的文件

```
poker-online/
├── server.js          服务器主程序
├── table.js           单桌游戏引擎
├── poker-core.js      扑克核心逻辑（牌型/胜率/AI/边池）
├── package.json       项目配置（声明依赖 ws）
├── .gitignore
├── public/
│   └── index.html     玩家打开的网页（前端）
├── test-core.js       测试（可选，不影响运行）
├── test-table.js      测试
└── test-server.js     测试
```

> ⚠️ **重要**：如果文件夹里有一个 `node_modules` 目录，**上传前请把它删掉**。
> 托管平台会自己重新安装依赖；带上它反而可能出错。

## 方式一：Railway（推荐，最简单）

Railway 支持长期运行的 WebSocket 服务，免费额度对朋友局完全够用。

### 第 1 步：装好准备工作
- 一个 GitHub 账号（没有就去 github.com 免费注册）
- 把 `poker-online` 文件夹传成一个 GitHub 仓库：
  - 登录 GitHub → 右上角 ➕ → New repository → 填个名字（如 `poker`）→ Create
  - 在仓库页面点 “uploading an existing file”，把 `poker-online` 里的文件**全部拖进去**（**不要**传 node_modules）→ Commit

### 第 2 步：在 Railway 部署
1. 打开 https://railway.app ，用 GitHub 账号登录
2. 点 **New Project** → **Deploy from GitHub repo** → 选你刚建的仓库
3. Railway 会自动识别这是 Node 项目，自动执行 `npm install` 和 `npm start`
4. 等 1–2 分钟，部署完成

### 第 3 步：拿到网址
1. 进入项目 → **Settings** → 找到 **Networking / Domains**
2. 点 **Generate Domain**（生成一个公网域名）
3. 得到一个网址，形如 `https://poker-production-xxxx.up.railway.app`

这就是你的游戏链接！发给朋友，大家打开就能建房/加入。

## 方式二：Render（同样免费）

1. 打开 https://render.com ，用 GitHub 登录
2. 点 **New** → **Web Service** → 连接你的 GitHub 仓库
3. 配置填写：
   - **Build Command**：`npm install`
   - **Start Command**：`npm start`
   - **Instance Type**：选 Free
4. 点 Create Web Service，等部署完成
5. 页面顶部就是你的网址，形如 `https://poker-xxxx.onrender.com`

> Render 免费版有个特点：**15 分钟没人访问会自动休眠**，下次第一个人打开要等 30–50 秒唤醒。朋友局影响不大。Railway 没有这个休眠（但免费额度有每月时长上限）。

## 怎么玩

1. 你打开网址 → 输昵称 → 选起始筹码 → **创建房间** → 得到一个 4 位房间号（如 `ABCD`）
2. 把房间号（或点“复制邀请”得到的文字）发到群里
3. 朋友们打开**同一个网址** → 输昵称 → 输房间号 → **加入房间**
4. 人齐了，房主选“补充电脑到几人”（人不够时让 AI 凑），点 **开始游戏**
5. 轮到谁，谁的屏幕下方就亮起操作按钮（弃牌/跟注/加注/全压），其他人看着
6. 一手结束自动开下一手

**手机也能玩**：iPhone / 安卓用浏览器打开网址即可，界面已适配竖屏。因为是网址不是文件，iOS 上不会再出现“只显示文字”的问题。

## 常见问题

**Q：免费够用吗？**
够。这游戏很轻，几个朋友玩消耗极小，在免费额度内。

**Q：朋友进不来 / 显示连接断开？**
确认大家用的是同一个网址；网址必须是 `https://`（平台默认就是）。WebSocket 在 https 下会自动用 wss，代码已处理。

**Q：有人手机切到后台再回来，画面没了？**
客户端会自动重连。如果牌局已经过了他的回合，他回来能看到最新状态。（断线期间轮到他会怎样属于第二阶段要打磨的：目前断线会让他离座。）

**Q：想改盲注/最大人数？**
盲注在 `poker-core.js` 顶部的 `SMALL_BLIND`/`BIG_BLIND`；最大人数 `MAX_SEATS`。改完重新部署即可。

**Q：本地先试一下行不行？**
行。在电脑上装好 Node.js，进 `poker-online` 目录，运行 `npm install` 再 `npm start`，然后浏览器开 `http://localhost:8080`。同一台电脑开几个浏览器标签就能模拟多人。
